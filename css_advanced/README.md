 Overview

This project is a static HTML landing page designed to practice page layout, semantic HTML, accessibility, and visual structure. The page includes:

A navigation bar

Hero section with a call-to-action button

Instructor profiles

Testimonial section

Popular tutorials

Membership benefits

Frequently Asked Questions (FAQ)

Footer with social media icons

It is intended as a learning exercise for building structured, multi-section web pages.

 Features

Fully structured multi-section layout

Semantic HTML elements for clarity and accessibility

Font Awesome icons for enhanced visuals

Sample images and placeholder logos

Organized instructor and tutorial cards

Clean footer with social links
